# Core & Curve プロジェクト決定事項ログ (Project Decisions Log)

このドキュメントは、本プロジェクトにおける主要なデザイン決定、技術仕様、および進捗状況を固定（FIX）し、今後の開発の基準とするための記録です。

## 1. デザインコンセプト
- **キーワード**: グラスモーフィズム (Glassmorphism), 静かなラグジュアリー, しなやかな強さ
- **カラーパレット**:
    - メイン背景: 深いチャコールグレー (`#242122`)
    - アクセントカラー: ピンク (`#e1a6b4`), ゴールド (`#dbc09e`)
- **タイポグラフィ**: 
    - メインフォント: `Shippori Mincho` (Google Fonts)
    - 見出し: `text-wrap: balance` を使用し、スマホでも美しい改行を維持

## 2. 主要機能
- **ファーストビュー (FV) アニメーション**:
    - 段階的なフェードイン (`fadeInUp`)
    - キャッチコピーのスワップ演出（「頑張る～」から「忙しくても～」へ）
- **Bikini Body 診断機能**:
    - JavaScriptによるインタラクティブなチェックリスト。
    - 3つのタイプ（S, R, B）に分類し、モーダルで結果を表示。
- **コンバージョン導線**:
    - グラスパネル調の公式LINE誘導セクション。
    - SNS（Instagram, note）への円形アイコンリンク。

## 3. インフラ・運用
- **構成**: HTML5 / CSS3 / Vanilla JS (No Frameworks)
- **デプロイ**: 
    - Git管理 (GitHub)
    - **Netlify** による自動デプロイ。`main` ブランチへのプッシュで即座に反映。
- **ディレクトリ構造**:
    - `index.html`: メインページ
    - `style.css`: 全デザイン定義（レスポンシブ対応）
    - `Core&Curve 原文 (1).md`: 素材・原文アーカイブ
